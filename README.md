# Landing Page — Operyx

Landing page développée avec React, TypeScript et Vite.

## Technologies

* React
* TypeScript
* Vite
* Tailwind CSS
* React Hook Form
* Zod
* i18n (FR / EN / DE / AR)

---

## Installation

Cloner le projet :

```bash
git clone https://github.com/BouoniManar/company-landing-page.git
```

Entrer dans le dossier :

```bash
cd company-landing-page
```

Installer les dépendances :

```bash
npm install
```

---

## Lancer le projet

Mode développement :

```bash
npm run dev
```

Puis ouvrir :

```text
http://localhost:5173
```

---

## Build production

Créer le build :

```bash
npm run build
```

Prévisualiser le build :

```bash
npm run preview
```
